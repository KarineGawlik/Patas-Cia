# Patas & Cia

Landing page developed for a pet shop.

🔗 Live project: https://patas-cia.vercel.app/

## Technologies

- HTML
- CSS
- Vercel

## Purpose

This project was created to present a pet shop, its services, products and contact information in a simple and responsive layout.

## Features

- About section
- Services section
- Products section
- Contact section
- Responsive layout
